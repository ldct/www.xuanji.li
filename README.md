# Deployment

- clone this repo, cd into it
- clone isicp
- cd into `isicp`, clone coding-js
- while in `isicp`, run render_all.sh
- cd into `xuanji.appspot.com`
- clone parallel-text
- clone js-scheme-stk
- clone goodreads-wall
- clone 2048
- clone card.js

- while still in `xuanji.appspot.com` folder: `appcfg.py update .`
